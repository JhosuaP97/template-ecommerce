# Next.js Teslo-shop APP

To run localy you needs database

```
docker-compose up -d
```

- -d means **detached**

- Mongo local URL

```
mongodb://localhost:27017/teslodb
```

## Set enviroment variables

Rename **.env.template** to **.env**

Install node modules and start Next

```
npm install
npm run dev
```
